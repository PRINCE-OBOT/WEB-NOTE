
### SVG Shapes

SVG has some predefined shape elements that can be used by developers:

1. Rectangle `<rect>`
2. Circle `<circle>`
3. Ellipse `<ellipse>
4. Line `<line>
5. Polyline `<polyline>
6. Polygon `<polygon>
7. Path `<path>
